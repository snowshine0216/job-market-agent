jma — Module Dependency Graph

Imports flow downward: cli → pipeline → sources / storage → domain (pure island). No back-edges.

Pure island invariant

  • • domain/ never imports sources/, storage/, pipeline/, or cli
  • • Frozen pydantic models — never mutate; use model_copy(update=…)
  • • Pure parsers (salary, location, experience) — no clock, no network
  • • Unit tests need no mocks

Factory + callbacks (pipeline.crawl)

  • • pipeline owns DB connection + URL cache
  • • source owns HTTP fetching
  • • on_fetch(url, status, blob_ref) writes to url_cache
  • • cache_get(url) reads with 24h TTL
  • • Decouples storage from sources, no circular imports

Adding a new source

  • • Drop config/sources/<name>.yaml
  • • Write a class implementing the JobSource Protocol
  • • Register a factory in cli.py
  • • Never raise on blockage — return SourceResult(status=BLOCKED, …)
